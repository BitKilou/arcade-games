# 🕹️ Kilou's Arcade

3 jeux d'arcade classiques entièrement codés en **vanilla HTML, CSS & JavaScript**. Aucune dépendance, aucun framework — juste du fun rétro.

## 🎮 Jeux disponibles

### 🐍 Snake
Guide le serpent, mange les pommes, grandis sans te mordre la queue.
- **Contrôles** : flèches directionnelles
- **Pause** : Espace

### 🧩 Tetris
Empile les pièces, complète des lignes, bats ton meilleur score.
- **Déplacer** : ← →
- **Tourner** : ↑
- **Descente douce** : ↓
- **Hard drop** : Espace
- **Pause** : P

### 🧱 Breakout
Casse toutes les briques avec la balle et la raquette. 5 niveaux avec difficulté progressive.
- **Déplacer la raquette** : ← →
- **Lancer la balle** : Espace

## 🛠️ Stack technique

- HTML5 Canvas
- CSS3 (animations, néon glow, scanlines)
- JavaScript vanilla (ES6+)
- Police : [Press Start 2P](https://fonts.google.com/specimen/Press+Start+2P)
- Zéro dépendance externe

## 📁 Structure

```
arcade-games/
├── index.html      ← Page d'accueil arcade
├── snake.html      ← Jeu Snake
├── tetris.html     ← Jeu Tetris
├── breakout.html   ← Jeu Breakout
└── README.md
```

## 📸 Features

- Thème rétro arcade avec effets néon et scanlines CRT
- Sauvegarde des meilleurs scores (localStorage)
- Ghost piece sur Tetris
- Briques multi-coups sur Breakout (niveaux avancés)
- Responsive design
- Fichiers 100% autonomes (CSS + JS inline)
